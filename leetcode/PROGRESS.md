## 学习笔记

### 核心模板

```
// 二分查找
left, right = 0, len(nums) - 1
while left <= right:
    mid = (left + right) // 2
    if nums[mid] == target: return mid
    elif nums[mid] < target: left = mid + 1
    else: right = mid - 1

// DFS
def dfs(node):
    if not node: return
    # 处理当前节点
    dfs(node.left)
    dfs(node.right)

// BFS
queue = [root]
while queue:
    node = queue.pop(0)
    # 处理当前节点
    if node.left: queue.append(node.left)
    if node.right: queue.append(node.right)
```
