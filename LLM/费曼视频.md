# LLM 费曼视频清单

每题 5 分钟，用白话讲，不准用术语。讲完回看，自己听不懂就重来。

## RAG

| # | 题目 | 一句话 |
|----|------|--------|
| 1 | RAG 是什么？ | 查资料再回答，不是凭空编 |
| 2 | 为什么要向量化？ | 文本比大小，语义相近靠得近 |
| 3 | 混合检索为什么比纯向量好？ | 关键词精确 + 语义模糊，互补 |
| 4 | 重排序做什么？ | 粗筛后再看一遍，挑最对的 |
| 5 | 父子块策略 | 小块查、大块看，各干各的 |

## Agent

| # | 题目 | 一句话 |
|----|------|--------|
| 6 | Agent 是什么？ | 能思考、能动手、能记事的 AI |
| 7 | Function Call 怎么工作？ | AI 说"我要调这个函数、用这些参数" |
| 8 | MCP 解决了什么问题？ | 统一插件接口，不用每个 AI 重写一遍 |
| 9 | A2A 是什么？ | AI 和 AI 之间的通信协议 |
| 10 | ReAct 模式 | 边想边做，做了再看结果 |

## 基础概念

| # | 题目 | 一句话 |
|----|------|--------|
| 11 | Transformer 注意力 | 每个词看全局，不是孤立理解 |
| 12 | Embedding 是什么？ | 把词变成数字坐标 |
| 13 | LLM 幻觉怎么来的？ | 概率采样，不是真的"知道" |
| 14 | 微调是干什么？ | 给通用模型补专业课 |
| 15 | 知识图谱 + RAG | 关键词搜不到的，从关系里找 |
