## LLM信息抽取任务介绍

- 首先，我们构造几个短文本对：

```JSON
1.('股票市场今日大涨，投资者乐观。', '持续上涨的市场让投资者感到满意。'),
2.('油价大幅下跌，能源公司面临挑战。', '未来智能城市的建设趋势愈发明显。'),
3.('利率上升，影响房地产市场。', '高利率对房地产有一定冲击。'),
```

- 我们期望模型能够帮我们识别出这 3 对句子中，哪几对描述的是相似的语言。

- 我们期望模型输出的结果为：

```Plain Text
['相似', '不相似', '相似']
```

## Prompt设计

- 在该任务的 prompt 设计中，我们主要考虑 2 点：

    - 需要向模型解释什么叫作「文本匹配任务」

    - 需要让模型按照我们指定的格式输出

- 为了让模型知道什么叫做「文本匹配任务」，我们借用 Incontext Learning 的方式，先给模型展示几个正确的例子：

```Plain Text
>>> User: 句子一: 公司ABC发布了季度财报，显示盈利增长。\n句子二: 财报披露，公司ABC利润上升
  >>> Bot: 是
  >>> User:句子一: 黄金价格下跌，投资者抛售。\n句子二: 外汇市场交易额创下新高
  >>> Bot: 不是
  ...
```



> 其中，`User` 代表我们输入给模型的句子，`Bot` 代表模型的回复内容。
> 
> 
> 
> 注意：上述例子中 `Bot` 的部分也是由人工输入的，其目的是希望看到在看到类似 `User` 中的句子时，模型应当做出类似 `Bot` 的回答。
> 
>

### 3\.2 编写文本匹配的指令

首先来分析这个指令包含哪些内容

1. 清晰的指令：告诉大模型完成金融行业的文本匹配任务

```Plain Text
现在你需要帮助我完成文本匹配任务，当我给你两个句子时，你需要回答我这两句话语义是否相似，你只能回到"是"或者"不是"，不能回答别的内容。
```

2. 输入数据分隔：根据输入数据和指定分类，需要明确输入数据的分隔符，通常用\&\#34;\#\#\#\&\#34;分隔

```Plain Text
下面是输入数据，以###作为分隔符，输入会包含两个句子。
###
%s
###
```



- 思考🤔：模型调用输入只有一个**content**参数，如何实现句子对输入

### 3\.3 开发文本匹配主函数

主函数的主要功能包括：

1. 指令拼接

2. 调用大模型输出结果

回到上面的思考问题，**两个文本如何拼接成一个content**，作为大模型的输入，**通过换行符拼接起来**。

```Plain Text
句子1: 公司ABC发布了季度财报，显示盈利增长。\n句子2: 财报披露，公司ABC利润上升
```

打印上面的内容为：

```Plain Text
print("句子1: 公司ABC发布了季度财报，显示盈利增长。\n句子2: 财报披露，公司ABC利润上升")
```



```Plain Text
句子1: 公司ABC发布了季度财报，显示盈利增长。
句子2: 财报披露，公司ABC利润上升
```



```Python
from model import model_chat

def sentence_pairs_match(sentence_pairs):
    """
    句子对匹配主函数
    :param sentence_pairs: 句子对，列表或者元组
    :return: 返回是否相似，是或者不是
    """
    if len(sentence_pairs) != 2:
        raise ValueError("length sentence_pairs not equal 2")

    prompt = """
    现在你需要帮助我完成文本匹配任务，当我给你两个句子时，你需要回答我这两句话语义是否相似
    下面是输入数据，以###作为分隔符，输入会包含两个句子
    ###
    %s
    ###
    你只能回到"是"或者"不是"，不能回答别的内容。
    """ % "\n".join(sentence_pairs)

    text = model_chat(content=prompt)
    if text not in ["是", "不是"]:
        text = "无法判断"
    return text


if __name__ == '__main__':
    sentence_pairs = [
        ('股票市场今日大涨，投资者乐观。', '持续上涨的市场让投资者感到满意。'),
        ('油价大幅下跌，能源公司面临挑战。', '未来智能城市的建设趋势愈发明显。'),
        ('利率上升，影响房地产市场。', '高利率对房地产有一定冲击。'),
    ]
    for each in sentence_pairs:
        print("输入数据为: ", each)
        print("匹配结果为: ", sentence_pairs_match(each))
```

结果输出为：

```Plain Text
输入数据为:  ('股票市场今日大涨，投资者乐观。', '持续上涨的市场让投资者感到满意。')
匹配结果为:  是
输入数据为:  ('油价大幅下跌，能源公司面临挑战。', '未来智能城市的建设趋势愈发明显。')
匹配结果为:  不是
输入数据为:  ('利率上升，影响房地产市场。', '高利率对房地产有一定冲击。')
匹配结果为:  是
```
